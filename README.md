# Spring JWT API

API REST avec authentification JWT (JSON Web Token) utilisant Spring Boot, Spring Security et MySQL.

 Technologies utilisées
 
Technologie	Version

Spring Boot	3.2.4

Spring Security	6.2.3

Spring Data JPA	3.2.4

MySQL	8.0

JJWT	0.11.5

Java	17

Maven	3.6+

# Fonctionnalités

    ✅ Authentification avec JWT

    ✅ Gestion des utilisateurs et rôles (ADMIN, USER)

    ✅ Mots de passe encodés avec BCrypt

    ✅ API REST stateless

    ✅ Protection des endpoints par rôle
